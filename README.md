**Text Summarizer using Gemini AI**

**Overview**

Text Summarizer is a simple AI-powered web application built with Streamlit and Google's Gemini API. The application accepts user-provided text and generates a concise summary along with key bullet points, making lengthy content easier to understand and review.

**Features**

  AI-powered text summarization
  Generates concise summaries
  Extracts important bullet points
  Simple and user-friendly interface
  Fast response using Gemini 2.5 Flash model
  Web-based application built with Streamlit
  
**Technologies Used**

  Python
  Streamlit
  Google Generative AI (Gemini API)
