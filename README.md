**Story Spinner - AI Story Generator**

**Overview**

Story Spinner is an AI-powered storytelling application built using Streamlit and Google's Gemini AI. The application generates engaging and creative stories based on a user-provided title or topic. Users can customize the story through genre selection, character preferences, and tone adjustments, making story creation fun and interactive.

**Features**

  Generate unique stories from a simple title or topic
  
  AI-powered content generation using Gemini 2.5 Flash
  
  Tone adjustment (Funny, Serious, Inspirational, Horror, etc.)
  
  Genre selection support
  
  Character-driven storytelling
  
  Structured output format : 
  
    Title
    
    Story Body
    
    Conclusion
    
    Simple and interactive user interface
    
    
**Technologies Used**

  Python
  
  Streamlit
  
  Google Generative AI (Gemini API)
